## Installation

Add this line to your application's Gemfile:

    gem 'anila-cli'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install anila-cli

## Usage

```bash
anila new my_awesome_project --version 1.0.0
```

### Update a project
```
cd ~/Sites/my_awesome_app
anila update --version 1.0.1
```

### Compile assets

```bash
cd ~/Sites/my_awesome_app
anila watch
```
